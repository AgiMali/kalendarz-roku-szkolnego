# Kalendarz roku szkolnego

Statyczna aplikacja webowa do planowania roku szkolnego nauczyciela.

## Zawartość

- widok miesięcy od września do czerwca
- plan tygodnia z lekcjami
- notatki organizacyjne do dni
- tematy i oznaczenia dla lekcji po kliknięciu dnia
- liczniki dni nauki i dni do końca roku szkolnego

## Uruchomienie

Otwórz plik `index.html` w przeglądarce.
